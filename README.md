## 📖 About The Project

**Hotel Stay Namaste** is a fully responsive hotel website designed to give guests a beautiful and seamless digital experience. The website showcases the hotel's rooms, amenities, photo gallery, and contact/booking information — all wrapped in a clean, modern design with a warm Indian hospitality touch.

Whether you're browsing on a phone, tablet, or desktop — the experience stays smooth and elegant across all devices.

---

## ✨ Features

- 🛏️ Room listings with details and pricing
- 🖼️ Beautiful photo gallery of the property
- 🏊 Amenities section highlighting hotel facilities
- 📞 Contact and booking information
- 📱 Fully responsive on all screen sizes
- 🎨 Clean, modern, and elegant UI
- ⚡ Fast and lightweight — no heavy frameworks
